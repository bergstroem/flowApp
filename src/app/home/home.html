Explore your Flow

What is Flow?

Flow is a simple tool that let's you log your users' usage flow in your application, and then visualize it.

SDK with ease

The Flow SDK is developed for simplicity and ease. A few lines of code in your application is all that is needed to get your flow started.

Web or mobile

Flow works seamlessly on both web or native mobile applications, there is no need to have a bunch of tools depending on platform.

1. Add the SDK

/* Boot the SDK */
var logger = flowSdk();

/* Add where page inits */
logger.enter('view1');

/* Add where page changes */
logger.navigate('view2', 'Click link');

2. API-response

{
  "label": "view1",
  "enter_probability": 1,
  "navigations": [
